## üìù **Important Code Patterns to Follow**

### **Database Connection:**
```php
require_once '../config/database.php';
// $pdo is now available
```

### **Require Login:**
```php
require_once '../config/database.php';
requireLogin(); // Redirects if not logged in
```

### **Prepared Statements:**
```php
$stmt = $pdo->prepare("SELECT * FROM users WHERE email = ?");
$stmt->execute([$email]);
$user = $stmt->fetch();
```

### **CSS Class Mapping:**
